# Contributing

Thanks for your interest in contributing! This repository is a personal sandbox for experiments in 2D physics and visualization â€” contributions are welcome, especially creative ideas and small experiments.

## How to contribute

- Share ideas: open an issue titled "Idea: ..." with a short description and motivation.
- Submit experiments: open a Pull Request for small, self-contained changes or features. Prefer small, well-scoped PRs.
- Report bugs: include steps to reproduce, the command you ran, relevant config, and any traceback.

## Focus areas

- Visualization experiments (rendering techniques, shaders, UI improvements)
- Physics tweaks (integrators, collision handling, performance improvements)
- Mini-games and interactive demos

## Collaboration & help

- If you want to collaborate, open an issue describing the idea and what you can contribute.
- Label issues with `collab` if you are seeking a co-author or pair programmer.
- For quick questions, open an issue or start a short PR with a prototype.

## Code of conduct

This is a friendly, inclusive project. Please be respectful. See `CONTRIBUTING.md` for how to get in touch if you experience issues.
